## üõ† Installation

1. Install dependencies

   ```sh
   cd portfolio-nextjs
   ```

   ```sh
   npm install
   ```

2. Start the development server

   ```sh
   npm run dev
   ```

## üöÄ Building the application

1. To generate a production build

   ```sh
   npm run build
   ```

1. Preview the site as it will appear once deployed

   ```sh
   npm run start
   ```
